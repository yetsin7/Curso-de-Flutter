# Documentacion oficial de Flutter y Dart

Flutter y Dart tienen documentacion oficial muy buena. Este archivo te deja la
ruta mas util para estudiar sin perder tiempo.

## Referencias principales

- [Flutter docs](https://docs.flutter.dev/)
  Flutter es el framework para construir interfaces multiplataforma.
- [Dart docs](https://dart.dev/guides)
  Dart es el lenguaje de programacion que usa Flutter.
- [Dart language tour](https://dart.dev/language)
  Es una guia mas directa sobre la sintaxis y los conceptos del lenguaje.

## Recursos clave para estudiar

- [Instalacion de Flutter](https://docs.flutter.dev/get-started/install)
  Primer paso para preparar el entorno.
- [Primer app con Flutter](https://docs.flutter.dev/get-started/codelab)
  Te ayuda a entender la estructura de una app real.
- [Widgets basicos](https://docs.flutter.dev/ui/widgets)
  Los widgets son los bloques con los que se construye la interfaz.
- [Gestion de estado](https://docs.flutter.dev/data-and-backend/state-mgmt/intro)
  El estado son los datos que cambian y afectan la UI.
- [Navegacion y rutas](https://docs.flutter.dev/ui/navigation)
  Explica como moverse entre pantallas.
- [Paquetes en pub.dev](https://pub.dev/)
  pub.dev es el registro oficial de paquetes de Dart y Flutter.

## Orden recomendado

1. Instala Flutter y valida el entorno.
2. Aprende primero Dart basico.
3. Construye interfaces con widgets y layouts.
4. Luego estudia estado, navegacion y consumo de datos.
